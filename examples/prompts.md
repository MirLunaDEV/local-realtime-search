# Example Prompts

Use these in LM Studio after enabling the `local_research` MCP tool.

## Current Facts

```text
Use local_research to answer: 오늘 날짜와 현재 한국 시간을 알려줘.
```

```text
Use local_research to answer: What is the latest LM Studio version and what changed recently?
```

```text
Use local_research to answer: 오늘 서울 날씨와 내일 비 올 가능성을 출처와 함께 알려줘.
```

## Local AI Research

```text
Use local_research to answer: Compare LM Studio MCP support with Open WebUI web search for local LLM users.
```

```text
Use local_research in deep mode to answer: What are the newest local reasoning models that run well around 9B parameters?
```

```text
Use local_research in deepsearch mode to answer: Compare the newest local reasoning model options around 9B parameters. Include source agreement, disagreements, uncertainty, and citations.
```

```text
Use local_research to answer: Find recent benchmark discussions for Qwen local reasoning models and summarize the tradeoffs.
```

## Product And Docs Tracking

```text
Use local_research to answer: What changed in the latest SearXNG docs around search API usage?
```

```text
Use local_research to answer: Is Crawl4AI still the best option for local page extraction? Compare recent alternatives.
```
